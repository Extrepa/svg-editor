## ‚ö†Ô∏è Warnings & Limits

### Path Count
- **Normal:** 10-500 paths
- **Warning:** 500-2000 paths (info message)
- **Error:** >2000 paths (warning message)

### Image Tracer
- Best for simple graphics
- Max 800px image size
- Complex photos = large files

### Path Simplifier
- Higher tolerance = more simplification
- May change appearance
- Start with 1-3 for subtle changes

## üí° Tips & Tricks

### Right-Click Color Picker
- Right-click any path to pick its color
- Automatically opens Colors editor
- Pre-fills color pickers

### Multi-Select
- Hold Shift/Ctrl/Cmd while clicking
- Select all paths in a group
- Edit multiple paths at once

### Layer Reordering
- Drag layers in Layers panel
- Top = appears on front
- Bottom = appears on back

### Path Simplification
- Use after Image Tracer
- Use before final export
- Check statistics to see reduction

### Performance
- Simplify complex paths
- Remove unnecessary paths
- Use Optimizer before export

## üêõ Troubleshooting

### Right-click not working
- Click directly on path, not background
- Ensure path has fill or stroke color

### Too many paths warning
- Use Path Simplifier
- Break into multiple SVGs
- Simplify traced images

### Performance issues
- Simplify paths
- Remove hidden paths
- Use Optimizer

### Colors not changing
- Check if path is selected
- Verify color format (#RRGGBB)
- Try "None" button for transparent
